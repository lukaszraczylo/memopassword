memopassword
============

Memorable password generator

### Example output

	Lukasz-Raczylo-MBP:memopassword nvm$ memopasswd 
	Generated password: 
	
	#%INcheD6 ( Based on: inched )
	
	Phonetic notation:
	# - hash
	% - percent
	I - capital-india
	N - capital-november
	c - charlie
	h - hotel
	e - echo
	D - capital-delta
	6 - six
